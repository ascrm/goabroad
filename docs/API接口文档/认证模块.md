### 认证模块接口文档（基于 `AuthController`）

#### 1. 用户注册（手机号注册）
- **路径**：`POST /api/auth/register`
- **请求参数（JSON Body）**
  - `phone` (string，必填)：手机号，正则 `^1[3-9]\d{9}$`
  - `code` (string，必填)：短信验证码，6 位数字
  - `password` (string，必填)：6-20 位密码
- **响应参数**
  - `code` (number)：业务状态码
  - `message` (string)：提示信息
  - `data` (object)
    - `token` (string)：当前登录 token
    - `tokenName` (string)：token 名称，来自 Sa-Token
    - `tokenTimeout` (long)：token 剩余有效期（秒）

#### 2. 用户登录
- **路径**：`POST /api/auth/login`
- **请求参数（JSON Body）**
  - `account` (string，必填)：手机号或邮箱
  - `password` (string，必填)：密码
- **响应参数**
  - `code` (number)
  - `message` (string)
  - `data` (object)
    - `user` (object)
      - `id` (long)
      - `username` (string)
      - `email` (string，可脱敏)
      - `phone` (string，可脱敏)
      - `nickname` (string)
      - `avatarUrl` (string)
      - `bio` (string)
      - `status` (`ACTIVE|INACTIVE|BANNED|DELETED`)
      - `createdAt` (datetime)
      - `updatedAt` (datetime)
    - `token` (string)
    - `tokenName` (string)
    - `tokenTimeout` (long)

#### 3. 用户登出
- **路径**：`POST /api/auth/logout`
- **请求参数**：无（需已登录，使用 Sa-Token 上下文）
- **响应参数**
  - `code` (number)
  - `message` (string)
  - `data`：`"登出成功"`（字符串）

#### 4. 发送短信验证码
- **路径**：`GET /api/auth/send-sms-code`
- **请求参数（Query）**
  - `phone` (string，必填)：接收验证码的手机号
- **响应参数**
  - `code` (number)
  - `message` (string)
  - `data`：`"验证码已发送，请查收短信"`

以上即 `AuthController` 中实际实现的四个接口的请求/响应要点，可直接并入 `docs/API接口文档/认证模块.md`。
