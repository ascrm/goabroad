### 用户模块接口文档（基于 `UserController`）

#### 1. 获取个人资料
- **路径**：`GET /api/user/profile`
- **请求参数**：无（需已登录）
- **响应参数**
  - `code` (number)：业务状态码
  - `message` (string)：提示信息
  - `data` (object)
    - `education_level` (string)：学历
    - `major` (string)：专业
    - `school` (string)：学校
    - `graduation_year` (number)：毕业年份
    - `gpa` (number|string)：GPA（如 `3.75`）
    - `toefl_score` (number)：托福分数
    - `ielts_score` (number|string)：雅思分数（如 `7.5`）
    - `gre_score` (number|string)：GRE 分数
    - `gmat_score` (number|string)：GMAT 分数
    - `work_years` (number)：工作年限
    - `current_company` (string)：当前公司
    - `current_position` (string)：当前职位
    - `location` (string)：所在城市
    - `birth_year` (number)：出生年份

#### 2. 更新个人资料
- **路径**：`PUT /api/user/profile`
- **请求参数（JSON Body）**
  - `education_level` (string，可选)：学历
  - `major` (string，可选)：专业
  - `school` (string，可选)：学校
  - `graduation_year` (number，可选)：毕业年份
  - `gpa` (number|string，可选)：GPA（如 `3.75`）
  - `toefl_score` (number，可选)：托福分数
  - `ielts_score` (number|string，可选)：雅思分数（如 `7.5`）
  - `gre_score` (number|string，可选)：GRE 分数
  - `gmat_score` (number|string，可选)：GMAT 分数
  - `work_years` (number，可选)：工作年限
  - `current_company` (string，可选)：当前公司
  - `current_position` (string，可选)：当前职位
  - `location` (string，可选)：所在城市
  - `birth_year` (number，可选)：出生年份
- **响应参数**
  - `code` (number)：业务状态码
  - `message` (string)：提示信息
  - `data` (object)：更新后的完整个人资料（字段同获取接口）
