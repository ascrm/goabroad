### 文件上传模块接口文档（基于 `FileController`）

#### 1. 通用文件上传
- **路径**：`POST /api/upload`
- **请求参数（Form-Data）**
  - `file` (MultipartFile，必填)：要上传的单个文件
- **响应参数**
  - `code` (number)
  - `message` (string)
  - `data` (string)：上传后文件的访问 URL

#### 2. 批量文件上传
- **路径**：`POST /api/upload/batch`
- **请求参数（Form-Data）**
  - `files` (MultipartFile[]，必填)：要上传的文件数组
- **响应参数**
  - `code` (number)
  - `message` (string)
  - `data` (array of string)：每个成功上传文件的 URL 列表

#### 3. 删除文件
- **路径**：`DELETE /api/upload`
- **请求参数（Query）**
  - `fileUrl` (string，必填)：需删除的文件 URL
- **响应参数**
  - `code` (number)
  - `message` (string)
  - `data`：`"文件删除成功"`（字符串）
