## 📊 分类设计方案

### 🏠 **首页 - 知识经验导向**
首页定位于**系统化的知识与经验学习**，帮助用户获取实用信息。

#### 推荐的顶部导航分类：

```javascript
// 首页 Tab 分类（知识经验导向）
const HOME_TABS = [
  { id: 'recommend', label: '推荐', icon: 'flame-outline' },
  { id: 'study', label: '留学', icon: 'school-outline' },    
  { id: 'work', label: '工作', icon: 'briefcase-outline' },  
  { id: 'visa', label: '签证', icon: 'card-outline' },       
  { id: 'life', label: '生活', icon: 'heart-outline' },      
  { id: 'guide', label: '攻略', icon: 'compass-outline' },   
];
```

**特点：**
- ✅ 偏向**干货型、教程型**内容
- ✅ 按**出国流程的关键节点**分类
- ✅ 强调**实用性和指导性**
- ✅ 适合用户**学习和查找资料**

## 📝 设计建议

### 对比分析：

| 维度 | 首页（知识经验） | 社区（日常生活） |
|------|----------------|-----------------|
| **内容形式** | 长文攻略、系统教程 | 短视频、图文动态 |
| **内容深度** | 深度、干货 | 轻松、真实 |
| **用户目的** | 学习、查资料 | 分享、交流 |
| **更新频率** | 相对稳定 | 高频更新 |
| **互动方式** | 收藏、点赞 | 评论、私信 |

### 实施建议：

1. **首页可以考虑：**
   - 攻略、申请、考试、签证、材料等**流程导向**分类
   - 或者按**时间阶段**：准备期、申请期、录取期、行前

2. **社区可以考虑：**
   - 推荐、关注、动态、Vlog、问答、海外等**内容形式+生活场景**分类
   - 强调**真实性**和**互动性**

3. **差异化要点：**
   - 首页：偏**工具属性**，帮助用户解决问题
   - 社区：偏**社交属性**，满足情感需求和归属感
