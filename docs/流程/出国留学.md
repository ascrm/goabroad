您好！从“出国工作”转为“出国留学”是规划上的重大转变。留学规划也同样需要详细的个人信息来确保计划的有效性和针对性。

为了给您生成一份**个性化且完整的出国留学计划**，我需要了解您的以下关键信息：

### 1. 个人背景与学术信息 (定位阶段)

| 序号 | 需要了解的信息 | 目的（为何需要这些信息） |
| :--- | :--- | :--- |
| **1.1** | **您目前的最高学历和专业（本科/硕士/其他）？** | 确定申请的起点，例如是申请硕士、博士还是本科。 |
| **1.2** | **您的平均成绩（GPA）是多少？** | 这是学校评估您的学术水平和竞争力的重要指标。 |
| **1.3** | **您是否有科研、实习或工作经验（如有，请简述）？** | 评估您的背景与目标专业要求的匹配度，影响文书准备。 |

### 2. 留学目标与方向 (目标阶段)

| 序号 | 需要了解的信息 | 目的（为何需要这些信息） |
| :--- | :--- | :--- |
| **2.1** | **您的第一目标留学国家或地区是哪里？** | 影响选校范围、申请系统、签证要求和预算制定。 |
| **2.2** | **您计划申请的专业领域或方向是什么？** | 这是选校和准备文书的核心，例如：CS、金融、艺术设计等。 |
| **2.3** | **您计划何时入学（具体到年份和学期，如：2027年秋季）？** | 确定倒计时时间表，规划标化考试和申请材料的准备时间。 |
| **2.4** | **您的留学预算范围是多少（学费+生活费总计）？** | 筛选学校和城市，有些学校/国家费用较高。 |

### 3. 标化考试与语言能力 (准备阶段)

| 序号 | 需要了解的信息 | 目的（为何需要这些信息） |
| :--- | :--- | :--- |
| **3.1** | **您是否有已取得的语言考试成绩？（如：托福/雅思成绩及分数）** | 评估您是否需要重新备考或只需要满足目标院校的最低要求。 |
| **3.2** | **您是否需要参加其他标准化考试？（如：GRE/GMAT/SAT/A-Level）** | 确定您需要准备哪些考试，影响您的备考计划。 |

### 4. 个人需求与偏好 (优化阶段)

| 序号 | 需要了解的信息 | 目的（为何需要这些信息） |
| :--- | :--- | :--- |
| **4.1** | **您毕业后是否有在当地就业或移民的打算？** | 影响选校策略（选择就业率高或有实习机会的专业）。 |
| **4.2** | **您对学校的类型或排名有什么偏好？** | 帮助我制定“冲刺、匹配、保底”的选校名单。 |

请您提供以上信息，我将为您量身定制一份详细的“出国留学规划”。
