### 1. 定居动机与目标国家 (确定移民路径)

| 序号 | 需要了解的信息 | 目的（为何需要这些信息） |
| :--- | :--- | :--- |
| **1.1** | **您的目标定居国家或地区是哪里？** | 确定移民政策、税务结构和居留要求。 |
| **1.2** | **您的定居主要目的是什么？** | 是为了子女教育、养老、追求更好的生活质量，还是为了资产配置？（这会影响移民方式的选择） |
| **1.3** | **您计划通过哪种方式实现定居？** | 是技术移民、投资移民、创业移民、家庭团聚，还是先工作/留学后转永居？ |
| **1.4** | **您计划何时启动定居流程，并期望何时完成落地？** | 确定整个时间表的紧迫性和规划节奏。 |

### 2. 个人背景与家庭情况 (评估资格与需求)

| 序号 | 需要了解的信息 | 目的（为何需要这些信息） |
