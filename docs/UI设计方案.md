- ✅ 扩展探索维度

### 🌟 **方案 3：4 Tab + 浮动按钮**（最推荐）

```
┌─────┬─────┬─────┬─────┐
│ 首页 │ 探索 │ 社区 │ 我的 │
```

**优势**：
- ✅ Tab 更简洁

- ✅ 视觉上更现代
- ✅ 参考：小红书、抖音

**劣势**：
- ❌ 规划入口不如 Tab 直观
- ❌ 可能被误认为只是发布按钮
